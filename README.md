# DemandSidePlatformII

Фронтенд часть cистемы  управления спроса для платформ по управлению предложением. 

## Установить проект
```
npm install
```

### Запуск проекта с HOT RELOAD
```
npm run serve
```

### Сборка проекта
```
npm run build
```

### Тестирование
```
npm run test
```

### Проверка через lint
```
npm run lint
```
